❓:函数 f(x)=-x^{2}+2 x 的单调递增区间为

🧠:嗯，我现在要找函数f(x) = -x² + 2x的单调递增区间。好，首先我得回忆一下怎么判断函数的单调性。记得应该是用导数的方法，对吧？求导之后，看导数的正负情况，导数大于零的时候函数递增，导数小于零的时候递减。那先来求这个函数的导数吧。原函数是f(x) = -x² + 2x，它的导数应该是f’(x) = -2x + 2。对吧？对的，因为导数公式里，x²的导数是2x，所以负x²的导数就是-2x，然后2x的导数是2，所以合起来就是f’(x) = -2x + 2。接下来，我需要确定导数什么时候大于零，也就是什么时候-2x + 2 > 0。解这个不等式的话，应该是把-2x移到右边，得到2 > 2x，然后两边都除以2，得到1 > x，也就是x < 1。对吗？是的，这时候导数就是正的，函数在x <1的时候单调递增。那什么时候导数小于零呢？当x >1的时候，-2x +2就会变成负数，所以函数在x >1的时候单调递减。这样的话，函数的单调递增区间应该是从负无穷到1，对吗？不过，这里我需要再检查一下，原函数是二次函数，开口向下的抛物线，顶点在哪儿呢？对了，二次函数ax² + bx +c的顶点横坐标是x = -b/(2a)。这里a是-1，b是2，所以顶点的x坐标是-2/(2*(-1)) = -2/-2 =1。对的，顶点在x=1的位置。那开口向下的话，顶点就是最高点，所以在顶点左边（x <1），函数应该是递增的，右边（x >1）递减的。这和导数的结果一致，所以没问题。那题目问的是单调递增区间，答案应该是区间（-∞，1）。不过，当x等于1的时候，导数为0，这时候是极值点，函数在这一点既不递增也不递减。所以开区间对吧？是的，所以应该写成区间形式的话，是(-∞, 1)。有没有哪里出错了呢？让我再检查一遍。导数计算没错，解不等式也没问题。二次函数的开口方向和顶点坐标都正确。看来没错。所以答案应该是x在负无穷到1之间的时候，函数单调递增。对了，有时候可能写成区间的形式需要注意符号，比如这里是用圆括号还是方括号。因为当x=1时导数为0，函数在这一点不单调递增，所以应该用开区间，也就是不包含端点。所以最终答案应该是(-∞, 1)。

🔑:函数 ( f(x) = -x^2 + 2x ) 的单调性可通过求导分析：1. 求导数： [ f'(x) = frac{d}{dx}(-x^2 + 2x) = -2x + 2 ]2. 解不等式判断符号： 当导数 ( f'(x) > 0 ) 时，函数递增： [ -2x + 2 > 0 implies -2x > -2 implies x < 1 ] 当导数 ( f'(x) < 0 ) 时，函数递减： [ -2x + 2 < 0 implies x > 1 ]3. 结合二次函数性质验证： 函数是开口向下的抛物线，顶点横坐标为 ( x = -frac{b}{2a} = 1 )。 开口向下时，左侧（( x < 1 )）递增，右侧（( x > 1 )）递减。4. 结论： 单调递增区间为 ( (-infty, 1) )，且 ( x = 1 ) 时导数为零，不包含端点。答案： boxed{(-infty, 1)}
