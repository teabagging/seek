❓:下列命题:(1) “若 mathrm{a} leq mathrm{b}, 则 mathrm{a}<mathrm{b} ” 的否命题;(2) “若 a=1, 则 a x^{2}-x+3 geq 0 的解集为 R ” 的逆否命题;(3) “周长相同的圆面积相等” 的逆命题;(4) “若 sqrt{2} x 为有理数, 则 x 为无理数” 的逆否命题.其中真命题序号为 ( )A. (2) (4)B. (1) (2) (3)C. (2)(3)(4)D. (1)(2)(3)(4)

🧠:好，我现在要解决这个题目，里面涉及到四个命题的真假判断，然后选择正确的序号组合。题目给出的选项是A到D，每个选项包含不同的命题序号组合。我需要逐一分析每个命题是否是真命题，然后根据分析结果选出正确的选项。首先，我需要明确每个命题的具体内容以及它们的否命题、逆否命题、逆命题等是什么样的，然后判断这些命题的真假。现在开始逐个分析：命题(1)：“若a ≤ b，则a < b”的否命题。首先，原命题是“如果a ≤ b，那么a < b”。但原命题本身是否成立呢？其实原命题并不总是真的，因为当a等于b的时候，即a ≤ b成立的情况下，a < b并不成立。比如，当a = b时，原命题的条件满足，但结论不成立，所以原命题是假命题。不过这里命题(1)不是原命题本身，而是它的否命题，所以需要明确否命题的结构。原命题的结构是“如果P，那么Q”，其否命题应该是“如果非P，那么非Q”。因此，原命题的条件P是“a ≤ b”，结论Q是“a < b”。那么否命题就是：“如果a > b，则a ≥ b”。这里需要注意，“非(a ≤ b)”即“a > b”，而“非(a < b)”即“a ≥ b”。因此，命题(1)的否命题是：“若a > b，则a ≥ b”。这个否命题是否为真呢？如果a > b成立的话，那么显然a ≥ b也一定成立，因为a已经比b大了，自然满足a ≥ b。所以这个否命题是真命题。因此命题(1)是正确的，它的否命题是真的。不过，这里可能有个误区，就是原命题的真假是否会影响其否命题的真假？实际上，原命题与其否命题的真假是独立的，也就是说，原命题为假的情况下，其否命题可能为真也可能为假，反之亦然。所以这里不要混淆。命题(1)的否命题是否正确，需要单独分析。我刚才分析得出，命题(1)的否命题是正确的，所以命题(1)是真命题。不过，这里需要再核实一下。原命题的否命题是否正确？原命题是“若P，则Q”，其否命题应该是“若非P，则非Q”。那么对于命题(1)，原命题是“a ≤ b ⇒ a < b”，它的否命题是“a > b ⇒ a ≥ b”。这里的结论是否正确？当a > b的时候，是否必然有a ≥ b？当然是，因为a > b已经满足a ≥ b了，所以这个结论是必然正确的。所以命题(1)的否命题是真的，因此命题(1)是真命题？不过，我之前说原命题本身是假的，比如当a = b时，原命题不成立，所以原命题是假的，但是它否命题是否是真的呢？是的，因为否命题的条件是a > b，此时结论是a ≥ b，而a > b的情况下，结论必然成立，因此否命题是真的。所以命题(1)是正确的。好的，那命题(1)是真命题，所以原题中的选项需要考虑是否包含(1)，比如选项B和D包含了(1)，所以可能正确。接下来分析命题(2)：“若a=1，则ax² - x + 3 ≥0的解集为R”的逆否命题。首先，原命题的条件是a=1，结论是ax² -x +3 ≥0的解集为实数集R。逆否命题的结构是“如果结论不成立，则条件不成立”。即原命题为“P → Q”，逆否命题就是“¬Q → ¬P”。所以原命题的条件P是“a=1”，结论Q是“ax² -x +3 ≥0的解集为R”。那么逆否命题应为：“如果ax² -x +3 ≥0的解集不是R，那么a ≠1”。接下来需要判断这个逆否命题是否正确。首先，原命题是否正确呢？当a=1时，方程变为x² -x +3 ≥0。二次项的系数是正数，判别式Δ = (-1)^2 -4*1*3 =1 -12 = -11 <0，所以这个二次函数开口向上，且无实根，因此对所有x∈R，x² -x +3 ≥0都成立，即解集为R。所以原命题是真命题，因此其逆否命题也是真命题，因为原命题与其逆否命题是等价的。因此命题(2)的逆否命题是真命题，即命题(2)是正确的。不过，这里是否有可能逆否命题本身是否正确呢？假设ax² -x +3 ≥0的解集不为R，即存在某个x使得这个不等式不成立，或者该不等式不是对所有x成立的。这时候需要推导出a≠1。原命题的条件是当a=1时解集为R，逆否命题就是当解集不为R时，a≠1。这等价于原命题的条件，所以原命题正确则逆否命题也正确。因为原命题正确，所以逆否命题正确，因此命题(2)是对的。接下来是命题(3)：“周长相同的圆面积相等”的逆命题。原命题的结构可以理解为：“如果两个圆周长相同，那么它们的面积相等。” 这是正确的，因为周长相同意味着半径相同，因此面积必然相同。原命题是真命题。那么逆命题是什么呢？逆命题是将原命题的条件和结论交换，即“如果两个圆面积相等，那么它们的周长相同”。这同样是正确的，因为面积相等的话，半径也相等，因此周长必然相同。所以逆命题也是真命题，因此命题(3)是对的。不过这里需要仔细检查是否存在漏洞。原命题的逆命题是否正确？原命题的条件是周长相同，结论是面积相等。逆命题的条件是面积相等，结论是周长相同。因为对于圆来说，周长C=2πr，面积S=πr²。面积相等的话，半径r相等，因此周长也必然相等。所以逆命题是正确的。因此命题(3)的逆命题是真命题，所以命题(3)是正确的。接着分析命题(4)：“若√2 x为有理数，则x为无理数”的逆否命题。原命题的结构是“若P，则Q”，即P是“√2 x是有理数”，Q是“x是无理数”。逆否命题应为“若非Q，则非P”，即“若x是有理数，则√2 x不是有理数”。现在需要判断这个逆否命题是否成立。也就是，如果x是有理数，那么√2 x是无理数吗？假设x是有理数，且不等于0，那么√2 x是否可以是有理数？假设√2 x是有理数，那么存在有理数q，使得√2 x = q，因此x = q/√2。但是x是有理数的话，那么q/√2也必须是有理数。然而，√2是无理数，有理数除以无理数（非零情况下）的结果是无理数，除非q=0，即x=0。如果x=0的话，原命题中的结论是“x是无理数”，但此时x=0是有理数，所以原命题的条件是“√2*0=0是有理数”，结论是“0是无理数”，显然原命题在x=0的情况下不成立，因此原命题是否成立呢？原命题是否存在反例？原命题是：“如果√2 x是有理数，那么x是无理数。” 当x=0时，√2 x=0是有理数，但x=0是有理数，因此原命题在这种情况下不成立，即原命题是假的。但是这里的逆否命题是否正确呢？原命题的逆否命题是：“如果x是有理数，那么√2 x是无理数。” 现在需要验证这个命题是否成立。当x是有理数时，如果x≠0，那么√2 x是无理数，因为有理数（非零）乘以无理数√2结果是无理数；当x=0时，√2 x=0是有理数。因此在x=0的情况下，逆否命题的条件是“x是有理数”（x=0确实是有理数），结论应该是“√2 x不是有理数”，即0不是有理数，但这显然不对，因为0是有理数。因此，在x=0时，逆否命题的条件成立，但结论不成立，所以逆否命题是假的。因此，命题(4)的逆否命题是假命题，所以命题(4)是错误的。但是这里可能存在误解，我需要再仔细分析原命题和逆否命题之间的关系是否正确。原命题是：“如果√2 x是有理数，那么x是无理数。” 其逆否命题应为：“如果x不是无理数（即x是有理数），那么√2 x不是有理数（即√2 x是无理数）。”现在，当x是有理数时，考虑√2 x是否为有理数。如果x为非零有理数，则√2 x是无理数，因为√2与x相乘结果仍然是无理数。但当x=0时，√2 x=0是有理数。因此，存在x=0的情况，x是有理数，而√2 x是有理数，因此逆否命题在这种情况下不成立，即存在反例。因此逆否命题是假的，所以命题(4)的逆否命题是假命题，因此命题(4)是假的。但是原命题是否真的为假呢？当x=0时，原命题的条件“√2 x是有理数”成立，但结论“x是无理数”不成立，因此原命题本身是假的。原命题的逆否命题的假也说明原命题是假的，但题目中的命题(4)是原命题的逆否命题是否为真，而实际上逆否命题在这里是假的，所以命题(4)是错误的，即命题(4)不是真命题。这与之前的分析一致，所以命题(4)的逆否命题是假命题，因此命题(4)是错误的，所以选项中不包括命题(4)。不过，我需要再次确认命题(4)的逆否命题是否正确。原命题的逆否命题是：“若x是有理数，则√2x不是有理数。” 但存在x=0的情况，此时√2x=0是有理数，所以逆否命题不成立。所以命题(4)的逆否命题是假的，因此命题(4)不正确，所以正确的命题不包括(4)。现在总结各个命题的真假：(1) 真；(2) 真；(3) 真；(4) 假。但根据之前的分析：命题(1)的否命题是正确的，所以命题(1)是真命题；命题(2)的逆否命题是正确的，所以命题(2)是真命题；命题(3)的逆命题是正确的，所以命题(3)是真命题；命题(4)的逆否命题是错误的，所以命题(4)是假命题。因此，正确的命题序号是(1),(2),(3)，即选项B：B.
