❓:试举一例, 说明“两个无理数的和仍是无理数” 是错误的: qquad

🧠:嗯，我现在得找一个例子来证明“两个无理数的和还是无理数”这个说法是不对的。就是说，我需要找出两个无理数，它们的和是有理数。对吧？首先，我应该回忆一下无理数的定义。无理数是不能表示成分数形式的数，也就是无限不循环小数，比如√2、π、e这些。而有理数就是可以写成两个整数相除的形式的数，比如1/2，3，或者0.5这样的。那问题来了，有没有可能两个这样的无理数相加之后变成一个有理数呢？比如说，如果我有一个无理数a，另一个无理数b，那么a + b等于一个有理数。这时候只要找到这样的a和b就可以了。这时候，我想到或许可以考虑互为相反数的例子。比如，假设a是一个无理数，那么b就是它的相反数，也就是-a。这时候，a + (-a) = 0，而0是有理数。但是这样的话，a和b都是无理数吗？假设a本身是无理数，那么-a当然也是无理数，因为乘以-1不会改变它是否能用分数表示。所以这样的话，他们的和是0，确实是一个有理数。比如，a是√2，那么b是-√2，它们的和就是0。这样是不是就符合条件了？不过，这样的话例子可能太简单了，或许题目希望找的是两个正的无理数相加得到有理数的情况？比如这样的情况是否存在呢？因为如果是正负相加的话，可能更直观。不过题目并没有说两个数必须都是正数，所以只要两个无理数相加得到有理数就可以。所以像√2和-√2的例子是成立的。不过，可能还有其他例子。比如说，如果一个无理数加另一个无理数得到的是整数或者分数。比如，π加上（1 - π），这时候和是1，而这里π是无理数，1 - π同样是无理数，因为如果1 - π是有理数的话，那么π=1 - 有理数，但有理数减1还是有理数，而π是无理数，所以矛盾。所以这里π和(1 - π)都是无理数，它们的和是1，有理数。这样也是一个例子。或者比如说，√3和(2 - √3)，它们的和是2。这里√3是无理数，2 - √3当然也是无理数，因为如果2 - √3是有理数的话，√3就等于2减去一个有理数，结果还是无理数，所以矛盾。这样他们的和是2，有理数。这同样满足条件。所以，这样的例子其实有很多，只要让两个无理数互为某种对称性，使得它们的和抵消掉无理部分，留下有理数即可。比如，选择任意一个无理数a，然后构造另一个数b = r - a，其中r是一个有理数。这样，a + b = r，而如果a是无理数，r是有理数，那么b也一定是无理数，因为有理数减去无理数结果是无理数。这样就能保证两个数都是无理数。比如，用√2作为a，然后r是1，那么b就是1 - √2。这样两个数都是无理数，相加得1。这样的例子也是可行的。总结一下，这样的例子需要满足两个条件：两个数都是无理数，它们的和是有理数。因此，构造的方式之一就是让其中一个数是某个有理数减去另一个无理数。或者直接使用相反数，比如√2和-√2的和是0，也是有理数。所以，问题要求的只是举一个例子，所以最简单的可能就是用√2和-√2相加的结果是0。或者用π和1-π相加得到1。或者像√3和2-√3相加得到2。这些例子都可以说明原命题是错误的，即存在两个无理数，它们的和是有理数，因此原命题不成立。

🔑:要反驳“两个无理数的和仍是无理数”，只需找到两个无理数相加结果为有理数的反例。例如：设第一个数为 sqrt{2}（无理数），第二个数为 -sqrt{2}（同样无理数）。它们的和为：[sqrt{2} + (-sqrt{2}) = 0]其中，0 是有理数。这表明两个无理数的和可以是有理数。类似地，取无理数 pi 和 1 - pi，其和为：[pi + (1 - pi) = 1]1 是有理数，再次验证结论。因此，原命题不成立。boxed{sqrt{2} + (-sqrt{2}) = 0}
