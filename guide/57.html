🔑:油气勘探技术涉及多种地球物理、地球化学和工程方法，不同技术会产生特定的检测数值，用于评估地下油气储层的特征。以下是具体示例：--- 1. 地震勘探技术- 反射波振幅异常值 - 数值示例：振幅强度 > 30 dB（分贝）的“亮点”或“暗点” - 应用：高振幅（亮点）可能指示含气砂岩，低振幅（暗点）可能与含油或孔隙流体有关。- 地震波速度 - 数值示例：砂岩含气时速度下降至 2,000–3,000 m/s（正常含水砂岩为 3,500–4,000 m/s）。 - 应用：通过速度异常识别含气层。- 频率衰减参数 - 数值示例：高频段（>50 Hz）信号衰减率 >30% - 应用：高频衰减可能与含气砂岩相关。- 各向异性系数 - 数值示例：裂缝性储层的各向异性系数 >10% - 应用：指示裂缝发育方向及密度。--- 2. 测井技术- 电阻率测井 - 数值示例：油层电阻率 20–200 Ω·m，水层电阻率 0.5–10 Ω·m - 应用：区分油、水层（油电阻率 > 水电阻率）。- 声波时差测井 - 数值示例：含气砂岩时差 >300 μs/m，致密砂岩 220–250 μs/m - 应用：计算孔隙度（高时差=高孔隙度）。- 自然伽马测井（GR） - 数值示例：泥岩 GR 值 100–150 API，砂岩 20–80 API - 应用：识别岩性（高 GR=泥岩，低 GR=砂岩）。- 中子孔隙度测井 - 数值示例：有效储层中子孔隙度 15–25% - 应用：评估储层孔隙发育程度。- 密度测井 - 数值示例：气层密度 <2.0 g/cm³，油层 2.2–2.4 g/cm³ - 应用：区分流体类型。--- 3. 地球化学勘探- 烃类地表浓度 - 数值示例：土壤中甲烷（CH₄）浓度 >10 ppm（背景值通常 <5 ppm） - 应用：指示地下油气微渗漏。- ΔC（热释碳酸盐）异常 - 数值示例：ΔC 含量 >2% - 应用：与油气藏相关的碳酸盐胶结物标志。- 碳同位素比值（δ¹³C） - 数值示例：甲烷 δ¹³C < -40‰（生物气）或 -25‰ 至 -35‰（热成因气） - 应用：区分天然气成因。--- 4. 地层测试技术（DST）- 压力恢复曲线斜率 - 数值示例：渗透率 1–1,000 mD（毫达西） - 应用：渗透率 >50 mD 为优质储层。- 试油产量 - 数值示例：产油 50–500 吨/天，产气 10⁴–10⁶ m³/天 - 应用：评价单井产能。- PVT分析（流体性质） - 数值示例：气油比（GOR） 100–500 m³/m³，API 重度 25–45° - 应用：确定原油品质（API >40° 为轻质油）。--- 5. 遥感与地表检测- 热红外温度异常 - 数值示例：地表温度较背景值高 5–10°C - 应用：烃类渗漏导致地表热异常。- InSAR形变监测 - 数值示例：地表垂向形变速率 >5 mm/年 - 应用：油气开采引起的地层压实。- 多光谱植被指数（NDVI） - 数值示例：NDVI 异常值 0.1–0.3（正常植被 NDVI >0.5） - 应用：烃类微渗漏导致植被胁迫。--- 总结这些数值需结合地质背景综合解释。例如： - 地震振幅异常 + 电阻率 >100 Ω·m + 低密度（2.0 g/cm³） → 高概率含气层； - ΔC >2% + 甲烷浓度 20 ppm + δ¹³C ≈ -50‰ → 生物气藏指示。 实际勘探中，多技术数据融合可显著提高预测精度。
