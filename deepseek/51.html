❓:圆 (x-3)^{2}+(y+2)^{2}=1 与圆 x^{2}+y^{2}-14 x-2 y+14=0 的位置关系是 (quad)A. 外切B. 内切C. 相交D. 相离

🧠:嗯，我现在要解决这个题目，关于两个圆的位置关系，选项是外切、内切、相交还是相离。题目给出了两个圆的方程，第一个是(x-3)^2 + (y+2)^2 = 1，第二个是x^2 + y^2 -14x -2y +14 = 0。好的，首先我需要回忆一下圆的一般方程和如何判断两个圆的位置关系。首先，圆的标准方程一般是(x - h)^2 + (y - k)^2 = r^2，其中(h, k)是圆心，r是半径。所以第一个圆已经给出标准形式，看起来可以直接读出圆心和半径。第二个方程是展开的一般式，可能需要配方来转化成标准形式。先处理第一个圆，方程是(x-3)^2 + (y+2)^2 = 1，所以圆心是(3, -2)，半径r_1 = sqrt{1} = 1。对吧，这里没问题。然后处理第二个圆的方程：x^2 + y^2 -14x -2y +14 = 0。我需要配方把它转化成标准形式。对于x的部分，系数是-14，所以配方的话是取-14的一半，即7，平方是49，然后加上再减去这部分。同样，y的部分系数是-2，所以一半是-1，平方是1，同样处理。所以配方过程：x^2 -14x + y^2 -2y = -14接下来，对x部分配方：x^2 -14x = (x -7)^2 -49对y部分配方：y^2 -2y = (y -1)^2 -1所以整个方程变成：(x -7)^2 -49 + (y -1)^2 -1 = -14合并常数项：(x -7)^2 + (y -1)^2 -50 = -14移项得：(x -7)^2 + (y -1)^2 = 36所以第二个圆的圆心是(7, 1)，半径r_2 = sqrt{36} = 6。好的，这样第二个圆的信息也清楚了。接下来，我需要计算这两个圆的圆心之间的距离，然后比较这个距离和两个半径的和、差，来确定它们的位置关系。圆心之间的距离公式是：d = sqrt{(x_2 -x_1)^2 + (y_2 -y_1)^2}这里第一个圆心是(3, -2)，第二个圆心是(7, 1)，所以代入计算：横坐标差：7 -3 =4纵坐标差：1 - (-2) =3所以距离d = √(4² +3²) = √(16 +9) = √25 =5现在，两个半径分别是r1=1，r2=6。它们的和是1+6=7，差是6-1=5。现在判断位置关系：如果圆心距离d > r1 + r2，那么相离；如果d = r1 + r2，外切；如果|r1 - r2| < d < r1 + r2，相交；如果d = |r1 - r2|，内切；这里，d=5，而r1 + r2=7，所以d=5 <7，同时，两个半径的差是5，也就是|6 -1|=5，所以d=5等于半径之差。这时候，这两个圆应该是内切的，对吗？所以选项B是内切，对吗？不过确认一下，是否还有其他可能性？比如，是否有可能是相交或者相离？因为当圆心距离等于半径之差时，两圆内切；当圆心距离在半径之差与半径之和之间时，相交；当圆心距离等于半径之和时，外切；当距离大于半径之和时相离，小于半径之差则是内含（这里可能没有选项）。但题目中的选项是外切、内切、相交、相离。所以当d=5，刚好等于6-1=5，这时候是内切，对应选项B。所以答案应该选B，内切。不过再检查一下计算是否正确：第一个圆心(3, -2)，第二个圆心(7,1)，距离计算：横坐标差：7-3=4，纵坐标差：1 - (-2)=3，所以√(4² +3²)=5，正确。第二个圆的半径是6，第一个是1，所以差是5，距离等于差，所以内切。没错，选项B。所以我认为正确选项是B，内切。答案选项 boxed{B}
